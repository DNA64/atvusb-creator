atvusb-creator

This program creates a patchstick from a USB thumb drive for the Apple TV 1 (1st Generation). This will NOT work on the newer generation (Black) Apple TV's.

About

This project was forked from https://github.com/davilla/atvusb-creator

FAQ

Q: An error occurred creating your patchstick. For more details visit: https://dna64.github.io/atvusb-creator/

A: If you get this error message when trying to build a patchstick it's because it the program was unable to fetch a required file. This could be because you don't have an internet location, or because the file no longer exists, has moved, or been renamed.
